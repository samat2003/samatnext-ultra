# Fast32 Frozen Inference Artifacts

This directory is generated by `scripts/save_fast32_frozen_artifacts.py`.

Frozen contents:

- `model_state.pt`: initialized Dynamic DNA-SSM model weights.
- `config.json`: exact Fast32 configuration that yields `216,320` trainable parameters.
- `architecture.json`: frozen architecture description and caveats.
- `benchmark_metadata.json`: benchmark, environment, and git metadata.
- `cached_precomposed_stateless_32.pt`: cached DNA params and master coefficient for the fastest 003G speed ablation.
- `cached_original_stateful_32.pt`: cached DNA params and initial state for the 003H architecture-preserving path.
- `SHA256SUMS.txt`: SHA256 checksums for frozen artifacts.

The fastest recorded result is `precomposed_stateless_32_fused_e2e` at `7.65 us` mean end-to-end full-logits latency, but it is an architectural speed ablation. The architecture-preserving result is `original_stateful_32` at `13.75 us` mean end-to-end full-logits latency.

No training was run as part of this freeze.
